# PRD – 個人記帳簿系統

## 1. 專案概述
### 背景與動機
現代人日常的收支管理分散於各種平台與紙筆，缺乏統一、可視化的工具。建立一個簡潔的個人記帳簿，讓使用者能在本機安全儲存與分析自己的財務狀況。

### 目標用戶
- 個人使用者（無論學生、上班族或自由業者）

### 核心價值主張
- 提供直觀的記帳、分類、預算分配與多帳戶管理功能，讓使用者能快速掌握財務流向，並可匯出資料作進一步分析或備份。

## 2. 功能需求
| 功能 | 使用者故事 |
|------|------------|
| **記帳** | 作為 **個人使用者**，我希望能 **快速新增收入或支出紀錄**，以便 **即時記錄我的財務變動**。
| **支出收入分類** | 作為 **個人使用者**，我希望 **自訂或使用預設的支出/收入類別**，以便 **在瀏覽紀錄時快速了解金錢流向**。
| **分配預算** | 作為 **個人使用者**，我希望 **為各類別設定月度預算**，以便 **在接近或超過預算時得到提醒**。
| **多帳戶管理** | 作為 **個人使用者**，我希望 **同時管理多個帳戶（如現金、銀行、信用卡）**，以便 **在一個介面查看全部資金總額**。
| **匯出資料** | 作為 **個人使用者**，我希望 **將記帳資料匯出為 CSV 或 Excel**，以便 **自行進行深入分析或備份**。

## 3. 非功能需求
- **技術限制**：使用 Flask + Jinja2 作為後端與前端框架，資料庫採用 SQLite，全部資料本地存儲。
- **效能**：頁面載入與操作回應時間 ≤ 200ms。
- **安全與隱私**：資料僅存於使用者本機，提供選項加密備份，避免外部洩漏。
- **可維護性**：程式碼遵循 PEP8、模組化設計，便於未來功能擴充。

## 4. MVP 範圍
| 等級 | 功能 |
|------|------|
| **Must Have** | 記帳、支出收入分類、多帳戶管理、匯出 CSV、基本預算設定與提醒 |
| **Should Have** | 圖表視覺化（支出比例、收入趨勢）、手機響應式版面 |
| **Nice to Have** | 多幣別支援、銀行自動匯入、資料雲端同步備份 |
